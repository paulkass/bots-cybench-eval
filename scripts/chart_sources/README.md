# Audited chart sources

These four PNGs are frozen copies of the paper's previously audited scaling figures. `../generate_charts.py` extracts only their colored trajectory geometry, then rebuilds axes, typography, legends, and layout in the shared Frontier style.

They remain the numerical authority because the canonical evaluation checkout used to generate the paper figures is not available in this workspace. Do not replace them with charts from a different checkout or newer run set without re-auditing the paper's quantitative claims and provenance.
